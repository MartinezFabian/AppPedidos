# AppPedidos
● Trabajo Práctico Integrador LABORATORIO DE COMPUTACIÓN II

Aplicación de gestión de pedidos donde se solicitan los siguientes datos:
- Nombre del cliente. 
- El pedido.
- Cantidad de unidades.

![1](https://user-images.githubusercontent.com/88947720/142082809-af3f7a0d-c8c0-4bcb-835d-4390e984ff05.png)

• Al hacer click sobre el botón "AÑADIR" se añaden los datos ingresados del pedido.

• Y se imprimen por pantalla de la siguiente forma:
![2](https://user-images.githubusercontent.com/88947720/142082816-d0a80cbf-5dbf-450b-af52-a9e6f27c4bed.png)

• Podemos repetir este proceso tantas veces como queramos:
![3](https://user-images.githubusercontent.com/88947720/142082819-7565e711-5a2d-40d4-8a5c-ca328e8b5ad5.png)
![4](https://user-images.githubusercontent.com/88947720/142082822-23234dac-02bd-4312-afdc-c8bef2afdea7.png)

• Esto se guarda de forma persistente en un archivo .txt : 
![5](https://user-images.githubusercontent.com/88947720/142082825-2787800e-53ad-43e5-8b8c-955ff2c51227.png)


• Podemos iniciar un nuevo pedido desde cero haciendo click sobre el botón "Iniciar nuevo pedido".
![6](https://user-images.githubusercontent.com/88947720/142082827-7ce3ab9e-12aa-49c1-a751-213b4e83c845.png)
![7](https://user-images.githubusercontent.com/88947720/142082830-2fe93d86-9bee-4b53-8cc4-80cb0a53c199.png)

⚠ Si intentamos añadir un pedido sin ingresar todos los datos requeridos obtendremos un mensaje de error:

![8](https://user-images.githubusercontent.com/88947720/142083920-761f8838-c8c5-4d42-bdfa-98c65dbd2a83.png)
